# Golden Dataset - QA Pairs для оценки поиска

Финальный проверенный датасет для оценки системы поиска информации по технологической инструкции "Очистка электролита от примесей".

## Содержание

- **qa_pairs_v2_podpunkti_golden.jsonl** — 85 вопросов и ответов в формате JSONL

## Статистика

| Метрика | Значение |
|---------|----------|
| Всего вопросов | 85 |
| Основные вопросы (qa_001-qa_060) | 60 |
| Вопросы об аббревиатурах (qa_abbr_001-qa_abbr_025) | 25 |

## Типы вопросов

### Основные вопросы (60)
- **scope** — область применения, назначение
- **fact** — факты, свойства
- **composition** — состав, параметры
- **equipment** — описание оборудования
- **operation** — операции, процессы
- **process** — технологические процессы
- **theory** — теория, пояснения
- **multi_hop** — вопросы, требующие информации из нескольких мест документа
- **rag_eval** — вопросы для оценки RAG системы
- **summary_multi_hop** — вопросы-резюме, требующие синтеза
- И другие специализированные типы

### Вопросы об аббревиатурах (25)
- **abbreviation** — расшифровка и пояснение аббревиатур
- **equipment_abbreviation** — аббревиатуры оборудования (ПКФ-80, ДТ-68-2,5)
- **chemical_abbreviation** — химические обозначения (H2SO4, Na2CO3, Cu, Co, Fe, Cl)
- **process_synonym** — названия переделов (медеочистка, железоочистка, кобальтоочистка)
- **safety_abbreviation** — безопасность (ПДК, аммиак)
- **troubleshooting_abbreviation** — неполадки и их решение
- **comparison_abbreviation** — сравнение (ПКФ-80 vs ПКФ-40)
- **abbreviation_context** — контекст использования сокращений
- **chemical_process_multi_hop** — многошаговые вопросы о химических процессах

## Структура JSON

```json
{
  "id": "qa_001",
  "question": "На какой процесс распространяется технологическая инструкция ТИ 3-48200234-05.1-12-2020?",
  "answer": "Инструкция распространяется на процесс очистки растворов...",
  "source_pages": [3],
  "source_chunks": [2],
  "multi_hop_required": false,
  "type": "scope",
  "aliases": ["Что значит ГМУ-1?", "Расшифруй ГМУ-1"]  // для вопросов об аббревиатурах
}
```

## Использование

Используйте этот датасет для:
1. **Оценки системы поиска** — Recall@1, Recall@3, Recall@5, MRR, NDCG@5
2. **Тестирования RAG систем** — проверка, находит ли система релевантные чанки
3. **Анализа качества поиска** — по типам вопросов, сложности (multi-hop), типам аббревиатур
4. **Бенчмаркирования** — сравнение разных моделей эмбеддингов и методов поиска

## Примеры использования

```python
import json

# Загрузка датасета
qa_pairs = []
with open("qa_pairs_v2_podpunkti_golden.jsonl") as f:
    for line in f:
        qa_pairs.append(json.loads(line))

print(f"Загружено {len(qa_pairs)} QA-пар")

# Группировка по типам
from collections import Counter
types = Counter(pair["type"] for pair in qa_pairs)
for type_name, count in types.most_common():
    print(f"{type_name}: {count}")
```

## Примечания

- Вопросы основаны на реальной технологической инструкции
- Source pages указывают номера страниц в исходном документе
- Multi-hop вопросы требуют синтеза информации из нескольких частей документа
- Aliases используются для альтернативных формулировок одного вопроса
